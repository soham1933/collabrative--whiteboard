# MERN Real-time Collaborative Whiteboard

Quick-start:

## Server
```bash
cd server
npm install
cp .env.example .env
# edit .env to set MONGO_URI if using Atlas
npm run dev
```

## Client
```bash
cd client
npm install
npm start
```

Server runs on PORT (default 8000). Client assumes server at http://localhost:8000 for dev.
